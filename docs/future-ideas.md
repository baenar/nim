# Future Variants & Experimental Mechanics

This document serves as a repository for brainstorming. There is a list of possible modifications to the game that we have considered but haven't yet implemented or fully explored. These ideas are meant to inspire further development and experimentation with the DraftSubtraction game.

---
## 1. Fog of War Variant
In this variant, players will not be able to see the opponent's subtraction set during the subtraction phase. So if an oppponent hasn't used one of his possible moves, the player will not see it. Potentially, we can add a mechanic, where a player can "reveal" the whole oppponent's set, sacrificing one of his moves. 

## 2. Dynamic Change of the Subtraction Set
Maybe we can add a mechanic, where a player can change one of the numbers in his subtraction set already during the subtraction phase, sacrificing one of the moves. For example, if a player has a subtraction set $S = \{2, 5, 7\}$, he can choose to change the number $5$ to $6$ during his turn, but in doing so, he would lose his current turn. This opportunity can be used once per game.
